# 👋 소개

안녕하세요.  
**Java / Spring Boot 기반 백엔드 개발자 정현식**입니다.

- 구조적이고 유지보수 가능한 서버 설계를 지향하며  
- Redis 기반 **실시간 처리**, WebSocket 기반 **실시간 매칭**,  
- JWT 인증/인가 시스템, JPA 성능 최적화,  
- Docker · AWS 배포 환경 구축 경험이 있습니다.

---

# 💡 관심 분야

- Redis 기반 실시간 서비스 (조회수 / 매칭 / 랭킹)
- 고트래픽에도 견딜 수 있는 서버 아키텍처 설계
- WebSocket(STOMP) 기반 실시간 통신
- JPA 성능 최적화 (N+1 해결 / 페이징 최적화 / 캐싱)
- JWT 인증 · Role 기반 권한 분리
- Docker + Nginx + GitHub Actions + AWS 자동 배포
